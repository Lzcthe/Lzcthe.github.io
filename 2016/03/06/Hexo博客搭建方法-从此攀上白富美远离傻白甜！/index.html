#Hexo 安装步骤详解

##准备工作
（此方法只适用于Windows）
先介绍一下hexo吧，hexo类似于Octopress、jekyll、Wordpress，是一种静态网页博客框架，我们只需要在电脑上搭建一个静态的博客框架，然后把它部署到github、gitcafe、coding登代码托管平台就可以搭建好自己的博客了。hexo的优点是速度快，免费，操作比Octopress简单，命令少，易于记忆。因为hexo是用Node.js搭建的博客平台，所以首先得下载安装Node.js。

快捷方法：

hexo g == hexo generate

hexo d == hexo deploy

hexo s == hexo server

hexo n == hexo new
